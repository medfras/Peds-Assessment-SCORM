# Stop the Bleed

Hemorrhage control is an escalation sequence, not a bucket sort. Match the intervention to the wound type and reassess after every control attempt.

## Escalation Pattern

- Expose the wound and use PPE.
- Apply firm direct pressure immediately.
- If bleeding persists, escalate based on wound location.
- Extremity hemorrhage: pressure/hemostatic dressing and tourniquet when appropriate.
- Junctional hemorrhage: wound packing and sustained pressure.
- Torso wounds: wound-specific management, breathing reassessment, and rapid transport.
- Reassess bleeding, distal perfusion, mental status, and shock signs.

## Common Traps

- Waiting for hypotension before treating major bleeding aggressively.
- Applying a standard tourniquet to a groin, neck, or torso wound.
- Forgetting to document tourniquet time.
