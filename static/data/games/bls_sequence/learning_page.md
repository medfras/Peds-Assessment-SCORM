# BLS Sequence

BLS is a rhythm of recognition, activation, CPR, AED analysis, shock/no-shock decision, and immediate resumption of compressions.

## Core Sequence

- Confirm unresponsive and not breathing normally.
- Check breathing and pulse for no more than 10 seconds.
- Activate help and apply AED.
- Start high-quality CPR.
- Stop motion for AED analysis.
- Shock if advised, then resume CPR immediately.
- If no shock is advised, resume CPR immediately.

## Pediatric Two-Rescuer Reminder

For a pediatric arrest with two rescuers, use 15:2 CPR until an advanced airway changes the ventilation strategy.

## Common Traps

- Long pauses for rhythm analysis or post-shock delay.
- Forgetting that AED application and compressions can happen in parallel with multiple rescuers.
- Restarting the sequence instead of continuing two-minute CPR/AED cycles.
